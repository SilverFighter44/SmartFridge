This project is for educational and portfolio purposes only and was created as a group assignment at university.

A simple mobile application built in Unity to help users keep track of food expiration dates in their refrigerator. The app makes it easy to monitor which products are still fresh, which are approaching their expiration date, and which are already past it. It’s a lightweight tool designed to reduce food waste and support smarter grocery planning.

The app originally included server‑based features for recognizing product names and expiration dates from photos.
These server features are currently non‑functional because the hosting has expired.
All core functionality remains available, and products can still be added manually.
